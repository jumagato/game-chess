# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Running the app

No build step. Open directly in a browser:

```bash
open index.html
# or serve it:
python3 -m http.server
```

## Architecture

The entire application is a single file: `index.html` (HTML + CSS + JS, no dependencies).

**State** is held in module-level `let` variables: `board` (8×8 string array), `turn`, `selected`, `validMoves`, `lastMove`, `history`, `capturedPieces`, `castlingRights`, `enPassantTarget`, `halfMoveClock`, `gameOver`.

Pieces are encoded as two-char strings: color prefix (`w`/`b`) + type (`K Q R B N P`), e.g. `'wQ'`, `'bP'`.

**Move generation** is two-stage:
1. `getRawMoves()` — pseudo-legal moves (geometry only, no check filtering)
2. `getLegalMoves()` — filters via `applyMove()` + `isInCheck()` to remove moves that leave the king in check

`applyMove()` is immutable — always returns a new cloned board.

**Undo** works by pushing a full state snapshot onto `history` before each move, then restoring it.

**Rendering** is a full re-render from scratch on every state change (`renderBoard()`, `renderSidebar()`, `updateStatus()`). No diffing.

**Algebraic notation** (SAN) is generated by `toAlgebraic()` before the move is applied; `+`/`#` suffixes are appended by `executeMove()` after evaluating the resulting position.

**3D board** is a CSS perspective/tilt effect (`#board-3d` wraps `#board`, which gets `rotateX(10deg)` + `preserve-3d`) — purely visual, no JS coordinate math involved, so clicks still resolve correctly against the tilted geometry.

**Theming** (dark/light) is driven by CSS custom properties on `:root`, overridden by a `[data-theme="light"]` block. `applyTheme()` sets/removes that attribute on `<html>`. Board square/piece colors are intentionally NOT themed — only page UI chrome.

**Audio** (move/capture sounds, ambient music) is synthesized via the Web Audio API in-script — no external audio files. `playSound('move' | 'capture')` is called from `executeMove()`. All three settings (theme, sound, music) persist via `localStorage`.
